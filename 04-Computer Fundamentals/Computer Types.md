## Types of Traditional Computers

| Computer Type | Screen & Keyboard | Main Purpose                                       |
| ------------- | ----------------- | -------------------------------------------------- |
| Laptop        | Yes               | Portable everyday computing                        |
| Desktop       | Yes               | Sustained performance at a fixed location          |
| Workstation   | Yes               | Professional and precision-based tasks             |
| Server        | No                | Provides services to multiple users over a network |

---

## Laptop

### Characteristics

* Portable
* Battery-powered
* Lightweight
* Designed for everyday tasks

### Common Uses

* Browsing
* Office work
* Studying
* Entertainment

---

## Desktop

### Characteristics

* Fixed location
* Better cooling
* More upgrade options
* Consistent performance

### Common Uses

* Gaming
* Office environments
* Home computing

---

## Workstation

### Characteristics

* High reliability
* Specialized hardware
* Precision-focused

### Common Uses

* Engineering simulations
* CAD design
* 3D modeling
* Scientific research

---

## Server

### Characteristics

* Runs continuously
* Serves multiple users
* Usually accessed remotely
* May not require a monitor or keyboard

### Common Uses

* Hosting websites
* Databases
* Cloud services
* File sharing

Sophia discovers that computers are hidden inside many everyday devices.

---

## Smartphone

### Description

A pocket-sized computer optimized for battery life, connectivity, and mobility.

### Examples

* iPhone
* Android devices

---

## Tablet

### Description

A touch-based portable computer with a larger screen than a smartphone.

### Examples

* iPad
* Drawing tablets

---

## IoT (Internet of Things) Devices

### Description

Network-connected devices designed for a specific purpose.

### Examples

* Smart thermostat
* Smart doorbell
* Fitness tracker
* Smart speaker

### Features

* Internet connectivity
* Remote control
* Data collection and reporting

---

## Embedded Computers

### Description

Computers built directly into another device.

### Examples

* Coffee machine controller
* Automatic door sensor
* Washing machine controller
* Lamp dimmer chip

### Features

* Performs a specific function
* Often invisible to users
* May not connect to the internet

---

## IoT vs Embedded Systems

| Feature              | IoT Device     | Embedded Computer         |
| -------------------- | -------------- | ------------------------- |
| Network Connectivity | Usually Yes    | Not Always                |
| Purpose              | Single-purpose | Single-purpose            |
| Internet Access      | Yes            | Often No                  |
| Examples             | Smart Doorbell | Coffee Machine Controller |

# 📝 Task 4: Why Computers Come in Different Flavors

Sophia asks:

> Why not just build one computer that does everything?

The answer is that every computer design involves trade-offs.

---

## Design Trade-Offs

### Mobility vs Performance

Portable devices:

* Lightweight
* Battery efficient
* Lower sustained performance

Examples:

* Smartphones
* Laptops

---

### Performance vs Cost

High-performance systems require:

* Better cooling
* Faster processors
* More memory

Examples:

* Workstations
* High-end desktops

---

### Reliability vs Cost

Servers and critical systems often include:

* Redundant power supplies
* Backup storage
* Fault tolerance mechanisms

These features increase reliability but also increase cost.

# 🔑 Key Takeaways

* Different computers are designed for different purposes.
* Laptops prioritize portability.
* Desktops prioritize sustained performance.
* Workstations prioritize precision and reliability.
* Servers provide services to multiple users.
* Smartphones are the most common pocket-sized computers.
* IoT devices connect to networks and exchange data.
* Embedded computers perform dedicated tasks inside larger systems.
* Every computer design involves trade-offs between performance, mobility, reliability, and cost.
